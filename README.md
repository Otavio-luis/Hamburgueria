Site de Hamburgueria:
Este projeto é um site simples de hamburgueria criado para mostrar uma promoção de combo de hambúrguer.

Sobre o projeto:
O site apresenta um combo promocional com visual moderno e responsivo.
Atualmente os botões de pedido não possuem funcionalidade, servindo apenas como demonstração visual.

Tecnologias utilizadas:
HTML
CSS
JavaScript

Responsividade:
O site funciona em celulares, tablets e computadores.
